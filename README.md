# Python Tutor 🐍

Интерактивное веб-приложение для обучения Python с ИИ-учителем «Питоша».

## Деплой на Render (бесплатно)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/sl-himmelreich/python-tutor-app)

После нажатия кнопки укажите переменную `GEMINI_API_KEY` — бесплатный ключ с [aistudio.google.com/apikey](https://aistudio.google.com/apikey).

## Возможности

- 6 уроков Python: переменные, типы данных, условия, циклы, функции, списки
- Теория → Тесты → Задания с редактором кода
- ИИ-учитель «Питоша» с озвучкой ответов
- Интерфейс полностью на русском языке
